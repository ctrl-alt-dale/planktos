===
PlanktOS (from *planktos*) is a lightweight OS-like firmware designed to run on an MCU (specifically ATmega328PB - yet) that uses an SSD1306 (128x64 Monochrome OLED), four TTP223 as Input, MB85RC256V as it's main Storage and Memory Module, and a BMP180 as an extraneous sensor for ambient environment pressure sensing and a crude ambient temperature sensor.

I implemented Display Management through the use of my own custom library for I2C Communication (or TWI).
